Manage Theme using CSS Vars

Introduction

This page provides a simple example on how you can manage multiple themes using CSS variables. This is useful if you're managing a small multi-theme web site.

Solution

The CSS variables are defined in the :root selector. We'll define the default and theme variables in two folders files:

default/variables.css

:root {
--font-family: Georgia, Cambria, san-serif;
--text-color: #000;
}

theme/variables.css

:root {
--font-family: 'Open Sans', sans-serif
--text-color: #666;
}

When the user selects a theme, the corresponding stylesheet is dynamically loaded.

<head>
<link ref="stylesheet" text="text/css" href="theme/variables.css">
<head>
