CSS Vars Example

Intro

This page provides a simple example on how you can manage multiple themes using CSS variables. The css-vars-ponyfill library allows support for legacy browsers (IE anyone 🙂).

** The theme stylesheets are inserted dynamically to the page and the css-vars-ponyfill re-compiles the styles.

Solution

The CSS variables are defined in the :root selector. We'll define the default and theme variables in two separate files:

default-variables.css

:root {
--font-family: Georgia, Cambria, san-serif;
--text-color: #000;
}

theme-variables.css

:root {
--font-family: 'Open Sans', sans-serif
--text-color: #666;
}

Then we'll add both stylesheets to the page in true cascading fashion.

<head>
<link ref="stylesheet" text="text/css" href="default-variables.css">
<link ref="stylesheet" text="text/css" href="theme-variables.css">
<head>
